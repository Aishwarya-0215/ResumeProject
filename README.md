# Resume Screening Project

## 📌 Description
This project is used to screen resumes using Python.

## ⚙️ Technologies Used
- Python
- NLTK
- Scikit-learn
- PyPDF2

## 🚀 How It Works
- Extracts text from PDF resumes
- Removes stopwords using NLP
- Compares resume with job description
- Gives similarity score

## ▶️ How to Run
pip install nltk scikit-learn PyPDF2
python main.py

## 📊 Output
Shows percentage match between resume and job description
